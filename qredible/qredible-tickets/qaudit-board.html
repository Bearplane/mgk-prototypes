Summary of new features (QP-3548):
1. Text search input (leftmost) — searches company name, address, document ID in real-time; highlighted blue border when active
2. State filter dropdown — filter by license issuing state (CA, CO, NV, OR, WA, etc.)
3. Days in Queue filter — bucket selector: 0–30 / 31–90 / 90+
4. Date Added range — From / To date pickers to filter tasks by when they were added to the queue
5. Filter chips in banner — each active filter shows as removable chip; individual ✕ or "Clear All"
6. Dimming behavior — non-matching cards dim to 35% opacity rather than disappearing; preserves column layout
7. Column counts update — count badge reflects only matching cards in filtered state
8. Export button — red/coral, right side; exports matching cards across all columns
9. Filter count badge — shows number of active filters (blue circle, toolbar right)
