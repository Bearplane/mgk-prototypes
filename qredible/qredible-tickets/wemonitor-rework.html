④ Filter Panel

Full Filter Panel — All Filterable Fields

Clicking ⚙ Filters opens a left-side panel (same pattern used for product-compliance filtering). Each row is an expandable section; selecting a primary value (e.g. an Industry Type, or Completed under Scan Status) reveals nested sub-filters — identical in shape to the existing Violative → Filter by violation type pattern. All six filters are multi-select; active selections appear as pills above the table (shown on the tabs above).

💡 Mirrors product-compliance behavior: selecting a parent row (Industry Type, or Scan Status = Completed) reveals a nested "Filter by…" sub-list — same pattern as Violative → violation type. Retailer can be nested under multiple industries (e.g. Hemp/CBD and Guns & Ammo) as independent selections.
Filterable Fields Summary
1
License Status
Multi-select: Active · Pending · Failed · Expired
2
Date Added
Range picker (presets + custom). Paired with a new Date Added column on both Active & Pending tables, sortable asc/desc by clicking the header.
3
Team Members
Multi-select with inline search. Includes an Unassigned option.
4
State
Multi-select with search bar (50 states + DC/territories).
5
Industry Type → Type of Company
Hierarchical multi-select (mirrors the Violative → violation type pattern). Select an industry type as a whole, or drill into specific types of company. Retailer can appear under both Hemp/CBD and Guns & Ammo as independent selections.
6
Scan Status
Multi-select: Not Scanned · Scanning · Completed. When Completed is selected, two optional date ranges appear — one for Product scan completed and one for Website scan completed.
+
Location search (separate)
In addition to the company search (Legal Name / DBA / Website / MID), a dedicated second search input accepts free-text city, state — matches the current WeMonitor UI's location field.
⑤ Industry Type Filter in Action

Filtered Views

💡 Industry Type + Type of Company display: Shown as paired pills with a simple separator dot (e.g., Hemp / CBD·Retailer). These appear as badges under the company's Legal Name — NOT a separate column. Each IT+ToC pair is one connected visual unit. Multi-industry companies show multiple pairs. Filtering works on either dimension independently.
