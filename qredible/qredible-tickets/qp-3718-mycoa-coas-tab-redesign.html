MyCOA COAs Tab — Unified Publish Toggle

The MyCOA COAs tab stays as it is today — same four summary cards, same filter bar, same columns. QP-3718 makes one targeted change: the separate MyCOA and Q-Report publish toggles collapse into a single unified toggle that controls both surfaces together.

MyCOA · COAs tab 1-to-1 with production layout Only structural change: Unified Publish

Problem

  • Two separate toggles (MyCOA & Q-Report) let them drift out of sync.
  • Merchants publish to MyCOA but forget Q-Report (or vice versa).
  • Row-level UI is crowded — two toggle controls per row.

Solution

  • One toggle per row — "Publish" — flips both MyCOA and Q-Report together.
  • No more unpaired state. Either both on or both off.
  • Everything else on the COAs tab is unchanged.
