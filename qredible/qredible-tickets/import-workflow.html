② Step 2b

Controlled Value Handling — Classification & Industry Type

Fields like Classification, Industry Type, and Type of Company are Qredible-controlled values. During import, users choose how to handle them.

✓
🤖 Auto-classify for me

Skip the classification column — our system will auto-classify products after upload using Q-Audit's keyword engine. Review and override anytime in the platform.

How it works: Product Definition text is matched against keyword rules (primary keywords, exclusion keywords, priority sort). Classification assigned automatically post-import.
✏️ I'll provide classification

Map a column from your file to Classification. Any value NOT in our accepted list will be flagged for you to correct before import.

Accepted values: Tinctures, Edibles, Flower, Topicals, Vape, Bath Bombs, Pets, Distillate, Dried Flower, Isolate
