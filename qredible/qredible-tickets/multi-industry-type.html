📊 Impact Summary — What Changes at Each Layer
Layer Before After
Data Model Single Industry Type + Type of Company per company (two columns) Array of IT+ToC pairs per company (junction table — many-to-many)
Sign-Up Single-select dropdowns — one pair at registration Multi-select with cascading Type of Company per checked industry
Pipeline One pair shown as flat pills Multiple pairs, visually grouped with dividers + color-coded per industry
Company Page Single "MERCHANT TYPE" header label Plural "MERCHANT TYPES" with grouped pills + Manage Types link
Bulk Upload Forced single IT+ToC for all rows in import Optional default multi-type + per-row override in column mapping step
Road to Qredibility Single questionnaire from one IT+ToC pair Aggregated questionnaire from all pairs — shared Qs deduplicated
Scan Profile One industry scan profile per company Union of all assigned industry scan profiles
MRI Profile One industry check set per company Union of all assigned industry MRI check sets
